Restaurant Inspections

Javascript web application that consumes data hosted by the Texas Parks and Wildlife ArcGIS Online Server. The application allows users to view the data associated with inspections of restaurants in the City of Austin. The application was written using the Twitter Bootstrap library, ESRI Javascript API, Dojo, jQuery, and jQuery UI. The data is hosted in ArcGIS Online and is consumed by the application as a Feature Service.

Capitol Image Viewer

This application was written in conjuction with Aubrey Drescher for a class we taught together to the Austin organization: "Supporting Women in Geography and GIS". Because the class was geared towards students who had limited exposure to Javascript or the Dojo framework, the application is very basic. What makes this application interesting, however, is the fact that it does not require an ArcGIS Server or a geodatabase. Everything the application needs is stored in the browser's memory and used when the application needs it. This makes the application very fast. The application was built using ESRI's Javascript API and the Dojo framework.
